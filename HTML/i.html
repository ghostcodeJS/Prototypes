Prototypes

em JavaScript

Os Protótipos são uma forma de compartilhar propriedades e métodos entre objetos em JavaScript

Esse poster foi inteiramente criada com CSS puro

É uma maneira eficiente de compartilhar código ao invés de criar várias cópias do mesmo código. O método sayHello é compartilhado por todos os objetos que são criados pela função construtora Person.
